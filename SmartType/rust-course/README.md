# Rust Course Roadmap

This repository tracks a full Rust learning journey from zero to pro. Each module has its own folder for code and notes. Start at Module 0 and work in order; add learnings and solutions as you go. Modules with code: `module0/hello_rust`, `module1/toolbox`.

## Modules

- `module0`: Setup and mindset, install the toolchain, and write your first program.
- `module1`: Rust basics: variables, control flow, strings, and vectors. Code: `module1/toolbox/`.
- `module2`: Ownership, borrowing, and lifetimes fundamentals. Code: `module2/rsgrep/`.
- `module3`: Structs, enums, and pattern matching. Code: `module3/password_checker/`.
- `module4`: Error handling and API design. Code: `module4/config_loader/`.
- `module5`: Traits, generics, and iterators. Code: `module5/stats_tool/`.
- `module6`: Modules, crates, and workspaces. Code: `module6/workspace_tool/`.
- `module7`: Testing, benchmarking, and tooling. Tests added for: `module2/rsgrep/`.
- `module8`: Smart pointers and interior mutability. Code: `module8/mem_db/`.
- `module9`: Concurrency basics. Code: `module9/parallel_scanner/`.
- `module10`: Async foundations. Code: `module10/async_crawler/`.
- `module12`: Unsafe Rust & FFI. Code: `module12/minimal_ffi/`.
- `module11`: Performance & memory demos. Code: `module11/optimizer/`.
- `capstone`: Starter port scanner. Code: `capstone/port_scanner/`.
